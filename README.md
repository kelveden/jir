jir
===

> Very much a work in progress...

A crude Jira CLI for those who want to avoid the UI.

Configuration
-------------

You'll need a file `~/.jirconfig` with the following structure:

```
{
  "user": "JIRA_USERNAME",
  "password": "JIRA_API_KEY",
  "host": "JIRA_HOST",
  "defaultBoard": <board-id>
}
```
